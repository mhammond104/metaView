# Sprint 6 — Workflow and UX Polish

This sprint completes the planned v0.3.1 feature work by improving access to existing functionality rather than adding another subsystem.

## Highlights

- Explorer-style thumbnail context menu.
- Two-click Compare With mode.
- Drag thumbnails directly onto tags.
- Multi-image rating.
- Centralised, non-conflicting keyboard shortcuts.
- Library and Research menu grouping.
- Search focus and context-sensitive rename shortcuts.

The proposed Compare-from-Preview workflow was deliberately not implemented.
