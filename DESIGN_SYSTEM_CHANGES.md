# v0.3.0 design-system refresh

- Replaced Catppuccin Mocha with Catppuccin Macchiato.
- Made Catppuccin Macchiato the default theme.
- Added automatic migration for users whose saved theme was Catppuccin Mocha.
- Added the Nord dark theme.
- Grouped dark and light themes in the Theme menu.
- Added centralised compact UI metrics in `src/metaview/ui_metrics.py`.
- Applied shared metrics to menus, controls, tabs, scrollbars, toolbars, and Preview positioning.
- Preserved all existing application behaviour and shortcuts.
