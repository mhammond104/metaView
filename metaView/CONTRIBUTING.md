# Contributing

Contributions and bug reports are welcome.

1. Fork the repository and create a branch from `main`.
2. Create and activate a Python 3.11+ virtual environment.
3. Install development dependencies with `python -m pip install -r requirements-dev.txt`.
4. Run `python -m compileall -q src tests` and `pytest` before opening a pull request.
5. Keep pull requests focused and describe the user-visible change clearly.

Please avoid committing generated build output, virtual environments, databases, or personal image collections.
