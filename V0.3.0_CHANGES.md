# metaView v0.3.0 candidate

## Theme system

Selectable from **Appearance > Theme** and applied immediately:

- Catppuccin Mocha (default)
- Gruvbox Dark
- Tokyo Night
- Dracula
- Catppuccin Latte
- Gruvbox Light

The selected theme is stored in QSettings and restored on launch. Styling is centralised in `src/metaview/theme.py` and applies to the main window, dialogs, menus, tooltips, lists, tables, scrollbars, and Preview overlay toolbar.

## Preview polish

- Floating overlay toolbar with fullscreen auto-hide
- T toggles/pins toolbar visibility
- Preview window geometry is remembered
- Windowed toolbar visibility is remembered
- Resolution is shown in the status area
- Fit, 100%, zoom, pan, fullscreen and result navigation remain available

## Earlier work included

- Restored thumbnail double-click opening
- Expanded thumbnail context menu
- Reveal in Explorer/Finder/File Manager
- Copy Path and Move to Trash
- Cached metadata hover tooltips
- Integrated reusable Preview window
