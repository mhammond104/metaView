# metaView v0.3.0

metaView v0.3.0 is a major workflow and interface release. It expands metaView from an image and metadata browser into a more complete environment for inspecting, comparing, organising, and documenting generative-AI image experiments.

## What's new

### Experiment Notebook

Create persistent notebooks and experiments, organise images into ordered runs, compare selected A/B generations, record run notes, and save overall conclusions. Existing experiments can now be opened directly from the main window or Experiment menu.

### Integrated Preview

Double-click a thumbnail or press Space to open a fast built-in viewer with smooth zoom, pan, fit, 100%, fullscreen, keyboard navigation, and a floating toolbar. The operating system viewer remains available as a separate command.

### Complete desktop workflow

The new File, Edit, Image, Experiment, View, and Help menus make the application's features easier to discover. Thumbnail context actions now include Preview, Open Image, Copy Path, Reveal in File Manager, and Move to Trash.

### Refined appearance

Version 0.3.0 introduces a compact visual design system and seven selectable themes:

- Catppuccin Macchiato
- Nord
- Tokyo Night
- Gruvbox Dark
- Dracula
- Catppuccin Latte
- Gruvbox Light

### Better at-a-glance metadata

Hover over a thumbnail to see its model, sampler and step count, scheduler, resolution, and LoRAs without leaving the browser.

## Upgrade notes

- Catppuccin Macchiato replaces Catppuccin Mocha. Existing Mocha preferences are migrated automatically.
- Ratings, Prompt Library entries, the image index, and Experiment Notebook data remain stored locally in the operating system's application-data directory.
- Extract downloaded release archives fully before running the application.

## Platforms

Pre-built archives are supplied for Windows, macOS, and Linux.
