# v0.3.0 UI density pass

This pass retains the bespoke v0.3.0 visual language while making metaView better suited to sustained desktop use.

- Reduced menu-bar and menu-row padding.
- Tightened toolbar and button spacing without reducing text size.
- Reduced input, tab and table-header height.
- Slimmed scrollbars and tightened group boxes.
- Reduced the main action-strip and prompt-bar margins.
- Made the floating Preview toolbar more compact.
- Tightened metadata/workflow panel spacing.

The aim is a professional desktop-tool density: compact, but not cramped.
